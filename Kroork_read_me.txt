"Кроссворд"
Выполнила Васильева Татьяна КН-103

Программа для заполнения кроссвода.

Способ запуска:
Kroork.py geometry.txt words.txt
справка: Kroork.py --help

На вход подаётся файл с геометрией, устроенной по типу:
***|*|*
+......
.**.*.*
+....**
где 
+ первая буква пересекающихся слов (одного из них)
| первая буква слова по вертикали
- первая буква слова по горизонтали
. любая другая буква
* пустое место

файл со словами:
баг, лето, код бегемот, газон

Девомпозиция:
Файл Kroork1.py управляет файлами parsefail.py и logic.py.
Первый из них обрабатывает входные данные,
второй решает кроссворд.
